misspy.Bot.notes#

notes#

ノート一覧を取得します。

Name

Type

Default

Description

local

bool

False

ローカルで作成されたノートのみを取得します。

reply

bool

True にすると返信だけを、 False にすると返信以外を取得します。値を設定しなければ返信であるかそうでないかに関係なくノートを取得します。

renote

bool

True にするとリノートだけを、 False にするとリノート以外を取得します。値を設定しなければリノートであるかそうでないかに関係なくノートを取得します。

withFiles

bool

True にすると添付ファイルのあるノートだけを、 False にすると添付ファイルがないノートだけを取得します。値を設定しなければ添付ファイルの有無にかかわらずノートを取得します。

poll

bool

True にすると投票を含むノートだけを、 False にすると含まないノートだけを取得します。値を設定しなければ投票の有無にかかわらずノートを取得します。

limit

int

10

取得するノートの最大数を指定します。

sinceId

str

指定すると、idがその値よりも大きいノートを返します。

untilId

str

指定すると、idがその値よりも小さいノートを返します。

notes_create#

ノートを作成します。返信やRenoteもこの関数で行います。

Name

Type

Default

Description

visibility

str
(public home followers specified)

public

ノートの公開範囲。

visibleUserIds

list
(str)

ノートを閲覧可能なユーザーのidのリスト。visibilityがspecifiedの場合のみ適用されます。

text

str | None

ノートの本文。

cw

str | None

ノートのCW。

localOnly

bool

False

Trueにすると、ローカルのみに投稿されます。

noExtractMentions

bool

False

Trueにすると、本文からメンションを展開しません。

noExtractHashtags

bool

False

Trueにすると、本文からハッシュタグを展開しません。

noExtractEmojis

bool

False

Trueにすると、本文から絵文字を展開しません。

fileIds

list
(str)

添付するファイルのid。

replyId

str | None

返信先のノートのid。

renoteId

str | None

Renote対象のノートのid。

channelId

str | None

投稿先のチャンネルのid。

poll

object | None

投票に関するパラメータ。

型#

Name

Type

Default

Description

choices

list

選択肢。

multiple

bool

False

Trueにすると、複数選択を許容します。

expiresAt

int | None

投票の締め切り。エポック秒で指定します。

expiredAfter

int | None

指定すると、ノート作成からexpiredAfter秒後に投票を締め切ります。expiresAtと併せて指定した場合、expiresAtが優先されます。
