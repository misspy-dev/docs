Welcome to misspy
=======================================

misspy is a Misskey API library for Python that supports Streaming API.

**Features:**

* ``async``/``await``\ syntax support
* Support for `StreamingAPI <https://misskey-hub.net/en/docs/api/streaming/>`__\
